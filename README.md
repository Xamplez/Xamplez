## Xamplez (Hackday #01 Zenexity) 

### Prerequisites

 * Play 2.1

### Notes

* For Github authentication, please use `http://localhost:9000` as main url (otherwise, redirectUri will faill)

### Contributes

You will need npm, grunt and bower. Then do:

- fork the project
- go to its root
- `npm install`
- `grunt bower`
- `grunt`
