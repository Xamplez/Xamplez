About Xamplez

Xamplez is the default instance of Xamplez following its manifesto.
It focuses on gathering & sharing your code snippets concerning XXX and everything about it.

How to?


Share a new Snippet

Fork the Root Gist & follow indications provided in gist's README


Search Snippet

You can type one or several terms separated by spaces. For example: "XXX YYY"

While typing, autocomplete will propose terms among known #tags.

You can go through autocomplete propositions using arrow UP/DOWN keys
